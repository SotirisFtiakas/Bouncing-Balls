# Bouncing-Balls
A simple Bouncing Balls project, using graphics.h 
__________________
### Input information
The first number is the number of balls.

For every line equal to the number of balls, there are 7 parameters.
* **1** = x coordinate (0,600)
* **2** = y coordinate (0,400)
* **3** = color (0,15)
* **4** = mass (0,10000)
* **5** = area (0,100)
* **6** = x speed (-100,100)
* **7** = y speed (-100,100)
